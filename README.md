Many Quizbowlers use carding (flashcards) to memorize tossups and their answers. 

This Flashcard generator parses the first line of each tossup and their respective answers from a TXT file and organizes them in a CSV file that can be used as flashcards. 

Directions:
1. Go to https://www.qbreader.org/db/
2. Input your categories, difficulties, etc.
3. Type in how many results you want
4. Click the blue txt button to download
5. Upload to the flashcard generator and enjoy!
